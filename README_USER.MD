# GrandBattleSupportTool ユーザー向けガイド

## 概要

GrandBattleSupportTool は、グランドバトルの拠点所持状況を日別に記録し、各ギルドの獲得ポイントを可視化するサポートツールです。  
MAP をクリックして拠点の所持チームを割り当てるだけで、6日間の累積ポイントを自動で集計できます。

## 機能

- **拠点所持の入力**
  - MAP 上の拠点をクリックして、選択中のギルドへ割り当て
  - 同じギルドを再クリックで解除、右クリックでも解除
- **日別管理**
  - 1日目〜6日目を切り替えて入力
  - 前日コピーで、前日の所持状態を当日に一括反映
- **ポイント表示**
  - 日別の所持拠点一覧を表で表示
  - 6日間合計ポイントをギルド別に表示
- **ギルド名編集**
  - A/B/C/D の名称を任意のギルド名に変更可能
- **保存・読込**
  - 一時保存（`save/owners.json.tmp`）
  - 保存管理画面から名前付き保存、読込、削除
- **GIF出力**
  - 日別の拠点状況をアニメーション GIF で出力
  - 各フレーム右下に「各ギルド累積PT」と「日数」を表示
  - 拠点マーカーはチーム別旗アイコン（`image/flag_a.jpg` など）を使用

## 操作方法

1. **起動**
   - アプリを起動します。
   - `image/map.jpg` と `nodes.json` が正しく配置されていることを確認してください。

2. **表示日を選択**
   - 画面上部の「表示日」で 1〜6 日目を選びます。

3. **入力チームを選択**
   - 「入力チーム（クリックで割当）」で対象ギルドを選択します。

4. **MAPで所持状況を入力**
   - 左クリック: 選択中ギルドに割り当て
   - 同じ拠点を同じギルドで再度左クリック: 解除
   - 右クリック: 解除

5. **必要に応じて前日コピー**
   - 2日目以降で「前日コピー」を押すと、前日の所持状態を当日に複製します。

6. **保存**
   - 「一時保存」: 作業途中データを簡易保存
   - 「保存管理」: 保存名を指定して保存、一覧から読込/削除

7. **GIF出力**
   - 「GIF出力」ボタンから出力先を選択して実行します。
   - 出力GIFには、拠点所持状況と日別の累積PTが含まれます。
