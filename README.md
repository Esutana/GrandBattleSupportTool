
# GrandBattleSupportTool ユーザー向けガイド

## 概要

GrandBattleSupportTool は、グランドバトルの拠点所持状況を日別に記録し、各ギルドの獲得ポイントを可視化する Windows Forms のサポートツールです。
MAP をクリックして拠点の所持チームを割り当てるだけで、1?6日目の累積ポイントを自動で集計します。

※ 本プロジェクトは .NET 8 (Windows) を対象にビルドされます。

## 機能

- 拠点所持の入力
  - MAP 上の拠点をクリックして、選択中のギルドへ割り当て
  - 同じギルドを再クリックで解除、右クリックでも解除
- 日別管理
  - 1日目?6日目を切り替えて入力
  - 「前日コピー」で前日の所持状態を当日に一括反映（変更フラグが立ちます）
- ポイント表示
  - 日別の所持拠点一覧を表で表示
  - 6日間合計ポイントをギルド別に表示
- ギルド名編集
  - A/B/C/D の名称を任意のギルド名に変更可能
- 保存・読込
  - 「一時保存」: 作業途中データを `save/owners.json.tmp` として保存
  - 「保存管理」: 名前付き保存（`save/*.json`）、上書き保存、読込、削除
  - 読込や保存で状態を読み込むと、編集フラグ（未保存フラグ）はクリアされます
- GIF出力
  - 日別の拠点状況をアニメーション GIF で出力
  - 各フレーム右下に「各ギルド累積PT」と「日数」を表示
  - 拠点マーカーはチーム別旗アイコン（`image/flag_*.png`）を使用
  - GIF 出力には `SixLabors.ImageSharp` 系のパッケージが必要です（プロジェクトに参照済み、ビルドシンボル `USE_IMAGESHARP` を使用）

## 操作方法

1. 起動
   - アプリを起動します。
   - 実行フォルダに `image/map.jpg`（MAP 画像）と `image/flag_*.png`（旗アイコン）、および `nodes.json` を配置してください。

2. 表示日を選択
   - 画面上部の「表示日」で 1?6 日目を選びます。

3. 入力チームを選択
   - 「入力チーム（クリックで割当）」で対象ギルドを選択します。

4. MAPで所持状況を入力
   - 左クリック: 選択中ギルドに割り当て
   - 同じ拠点を同じギルドで再度左クリック: 解除
   - 右クリック: 解除

5. 必要に応じて前日コピー
   - 2日目以降で「前日コピー」を押すと、前日の所持状態を当日に複製します（この操作で未保存フラグが立ちます）。

6. 保存
   - 「一時保存」: `save/owners.json.tmp` に一時保存（通常の保存操作）。
   - 「保存管理」: 保存一覧ダイアログで名前を付けて保存、一覧から選んで「上書保存」や「新規保存」、読み込み、削除が可能です。

7. GIF出力
   - 「GIF出力」ボタンから出力先を選択して実行します。
   - ImageSharp を利用して GIF を作成します（プロジェクトが ImageSharp を参照している必要があります）。

## 保存ファイルの配置

- 一時保存: `save/owners.json.tmp`
- 名前付き保存: `save/{timestamp}_{name}.json`

## 開発/ビルド注意点

- .NET 8 Windows Forms アプリとしてビルドしてください。
- `image/map.jpg`, `image/flag_*.png`, `nodes.json` は実行時に出力フォルダへコピーされる設定になっています。
- GIF 出力のために `SixLabors.ImageSharp` 系パッケージをプロジェクトに追加（本リポジトリでは既に PackageReference があります）し、ビルドシンボル `USE_IMAGESHARP` が有効になっています。
